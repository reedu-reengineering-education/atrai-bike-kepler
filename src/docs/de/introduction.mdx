# Einführung

Willkommen beim Projekt!  
Diese Anwendung verwendet **[Kepler.gl](https://kepler.gl/)**, um Geodaten direkt im Browser zu visualisieren.  
Mit Kepler.gl können Sie Datensätze hochladen, interaktive Karten erkunden und ansprechende Visualisierungen erstellen – ganz ohne Programmierkenntnisse.

## Hauptfunktionen

- 📍 CSV-, JSON- oder GeoJSON-Dateien hochladen
- 🗺️ Kartendaten interaktiv filtern und erkunden
- 🎨 Kartenstile und Layer anpassen
- 📤 Karten exportieren und teilen

## Für wen ist das gedacht?

Dieses Tool richtet sich an:

- Datenanalysten
- Forscher
- Alle, die mit standortbasierten Daten arbeiten

# So laden Sie Ihre eigene CSV-, JSON- oder GeoJSON-Datei in Kepler.gl hoch

Das Hochladen Ihrer Daten in Kepler.gl ist schnell und einfach.

## Schritte

1. **Öffnen Sie die Anwendung** in Ihrem Browser.
2. **Klicken Sie auf** die Schaltfläche „Layer“.
3. **Klicken Sie auf** die Schaltfläche „Daten hinzufügen“.
4. **Wählen Sie Ihre Datei** von Ihrem Computer aus.
5. **Warten Sie, bis die Daten geladen sind** – die Karte wird automatisch aktualisiert.
6. **Passen Sie** die Kartenebenen und -stile nach Bedarf an.

# 📖 Einführung

Willkommen zum Projekt!  
Diese Anwendung erweitert **[Kepler.gl](https://kepler.gl/)**, um Geodaten direkt im Browser zu visualisieren.

Mit Kepler.gl können Sie ganz einfach Datensätze hochladen, interaktive Karten erkunden und schöne Visualisierungen erstellen – ohne eine einzige Zeile Code zu schreiben.

## ✨ Hauptfunktionen

- 📍 CSV-, JSON- oder GeoJSON-Dateien hochladen
- 🗺️ Interaktive Filterung und Erkundung von Kartendaten
- 🎨 Anpassung von Kartenstilen und Layern
- 📤 Karten exportieren und mit anderen teilen
- 🗂️ Zugriff auf **vorgefertigte Karten** für:
  - Gefahrenzonen
  - Luftverschmutzung
  - Unebene Straßen
  - Überholabstände
  - Geschwindigkeitskarten
  - Verkehrsfluss
- 📊 Ansicht von **Statistiken verschiedener Kampagnen**, einschließlich:
  - Wiesbaden
  - Münster
  - São Paulo
  - Arnsberg
  - Greifswald

## 👥 Für wen ist dieses Tool?

Dieses Tool richtet sich an:

- Datenanalysten
- Forscher
- Alle, die mit standortbasierten Daten arbeiten

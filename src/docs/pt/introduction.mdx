# 📖 Introdução

Bem-vindo ao projeto!  
Esta aplicação estende o **[Kepler.gl](https://kepler.gl/)** para visualizar dados geoespaciais diretamente no navegador.

Com o Kepler.gl, você pode facilmente fazer upload de conjuntos de dados, explorar mapas interativos e criar visualizações bonitas sem precisar escrever código.

## ✨ Principais Funcionalidades

- 📍 Fazer upload de arquivos CSV, JSON ou GeoJSON
- 🗺️ Explorar e filtrar dados do mapa de forma interativa
- 🎨 Personalizar estilos e camadas do mapa
- 📤 Exportar mapas e compartilhar com outras pessoas
- 🗂️ Acessar **mapas pré-carregados** para:
  - Zonas de perigo
  - Poluição do ar
  - Estradas esburacadas
  - Distâncias de ultrapassagem
  - Mapas de velocidade
  - Fluxo de tráfego
- 📊 Visualizar **estatísticas de diferentes campanhas**, incluindo:
  - Wiesbaden
  - Münster
  - São Paulo
  - Arnsberg
  - Greifswald

## 👥 Para quem é esta ferramenta?

Esta ferramenta é destinada a:

- Analistas de dados
- Pesquisadores
- Qualquer pessoa que trabalhe com dados baseados em localização

# 📖 Introduction

Welcome to the project!  
This application extends **[Kepler.gl](https://kepler.gl/)** to visualize geospatial data directly in your browser.

With Kepler.gl, you can easily upload datasets, explore interactive maps, and create beautiful visualizations without writing any code.

## ✨ Key Features

- 📍 Upload CSV, JSON, or GeoJSON files
- 🗺️ Explore and filter map data interactively
- 🎨 Customize map styles and layers
- 📤 Export maps and share with others
- 🗂️ Access **preloaded maps** for:
  - Danger zones
  - Air pollution
  - Bumpy roads
  - Overtaking distance
  - Speed maps
  - Traffic flow
- 📊 View **statistics from different campaigns**, including:
  - Wiesbaden
  - Münster
  - São Paulo
  - Arnsberg
  - Greifswald

## 👥 Who is this for?

This tool is designed for:

- Data analysts
- Researchers
- Anyone working with location-based data
